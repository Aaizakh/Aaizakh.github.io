To-doer

A Google Chrome Extension that helps users manage their tasks better by staying more organized

Problem

Reflecting back on how to manage my time better when classes start again, the biggest issue I faced duruing my school term was keeping my dailiy tasks organized and listed. While talking to some of my friends from who also started university recently, they mentioned the same issue. They also mentioned that they're not too fond of Task-Management applications as most of them have so many features that it becomes over-whelming for them to actually use the apps on a daily basis

Goal

To work towards fixing this issue, my main objective was to come up with a solution that's very user-friendly and easy to use. My goal wasn't to include fancy features that send notifications on your phone if you don't finish your tasks on time. Rather, i wanted to design and develop a straight-forward application that just tracks what tasks you have to do and categorizes them under different projects

Result

After switching between a Webpage, Web Application and a Chrome extension, I decided to go develop a Chrome extension. In this situation, a chrome extension seemed to be the best solution as it would be readily accessible ~ literally a click away. As soon as users would open a new tab, they will have an interactive page where they can add/name projects and add tasks underneath.

To design the UI, I used Figma for designing wireframes and live prototypes of the application. Later on, I developed it using HTML, CSS, JavaScript and jQuery

You can view the code at: https://github.com/Aaizakh/To-doer
